# Based on theme
https://github.com/pages-themes/modernist?tab=readme-ov-file

# Install github-pages gem
https://rubygems.org/gems/github-pages
```
gem install github-pages
```

# Preview locally

## Pre-requisites
```
gem install github-pages -v 231
gem install wdm
gem install bundler
bundle install
bundle add webrick
```

## Run locally
```
bundle exec jekyll serve
```
